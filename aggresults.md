There is one dropdown menu automatically detecting the classification designs that have been run. The user can select which design to explore, and then click on the Load button to update the figures.
All graphs can be saved, and will be saved by default in SVG format. This can be changed in the `AggregatedResultsTab.py` file, at the beginning of the file.
